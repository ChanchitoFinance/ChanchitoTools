# Google Pay Book Store - Testing Integration

Internal testing project for Google Pay payment gateway integration.

## Important

- This is for **INTERNAL TESTING ONLY**
- $1 per book for testing purposes
- Not a commercial website

## Setup

1. Install dependencies:

```bash
npm install
```

2. Configure your `.env` file with your Google Pay Merchant ID

3. Run development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Google Pay Configuration

1. Get your Merchant ID from Google Pay Business Console
2. Configure in `.env` file
3. For production, change environment from 'TEST' to 'PRODUCTION' in code
